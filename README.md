Rust言語だぜ☆（＾～＾）　開発している最中のまま投げ込むぜ☆（＾～＾）

## Compile

Visual Studio Code `[Terminal] - [New Terminal]`.  

```Shell
cargo build --release
```

## Start on the Shogidokoro

Shogidokoro `[対局(G)] - [エンジン管理...] - [追加...]`  
`./target/release/kifuwarabe_shogi2018.exe`

## Start on the terminal

Visual Studio Code `[Terminal] - [New Terminal]`.  

```Shell
cargo run main.rs
```
